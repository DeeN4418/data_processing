1. в отдельной директории разместить исполняемый файл treat_file.py и папку 'data', в которой будут размещены исходные файлы с данными для построения ВАХ (это сырые файлы, полученные с характериографа)
2. (опционально) запускаем оболочку IDLE Shell,  и открываем в ней файл treat_file.py
3. (опционально) нажимаем F5 происходит запуск программы и начинается обработка файлов
4. в директории с программой появляется папка results, в которой находятся файлы с очищенными данными и итоговый файл resfile.txt
	- файлы с добавлением _clr - данные, очищенные от нулей в начале, но с пробями в конце
	- файлы с добавлением _wbd - из файла _clr с удаленными пробоями
	- в файле resfile.txt находится информация, выбранная из файлов: 
		filename - имя обработанного файла
		Utrh - напряжение начала эмиссии
		Umax - максимальное допробойное напряжение
		deltaU - диапазон рабочих напряжений 
		Imax - максимальный допробойный ток
		a - коэффициент из уравнения экспоненциальной зависимости
		k - показатель степени из уравнения экспоненциальной зависимости (крутизна ВАХ)
